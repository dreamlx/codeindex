# codeindex 2025 Q1 Roadmap

## Vision
成为 AI 原生的代码索引工具，为大型代码库提供智能化的文档生成和代码理解能力。

## 2025 Q1 目标

### Epic 1: 核心功能完善
- [x] Python 代码解析器（基于 tree-sitter）
- [x] CLI 基础命令（scan, init, status, list-dirs）
- [x] 外部 AI CLI 集成
- [x] 配置系统（.codeindex.yaml）
- [ ] 测试覆盖率 ≥ 80%
- [ ] 性能优化（大型项目扫描）

### Epic 2: 多语言支持
- [ ] TypeScript/JavaScript 解析器
- [ ] Java 解析器
- [ ] Go 解析器
- [ ] 语言自动检测

### Epic 3: MCP 服务集成
- [ ] 设计 MCP 服务协议
- [ ] 实现索引查询接口
- [ ] 支持 Claude Code 集成
- [ ] Skill 系统（/sc:index-repo, /sc:arch-query）

### Epic 4: 智能索引优化
- [ ] 增量索引（只扫描变更文件）
- [ ] 缓存机制
- [ ] 索引压缩存储
- [ ] 并行扫描优化

## 里程碑

- **v0.2.0** (2025-01-31): 多语言支持 + 测试覆盖
- **v0.3.0** (2025-02-28): MCP 服务集成
- **v0.4.0** (2025-03-31): 智能索引优化 + 性能基准

## 成功指标

- ✅ PyPI 包发布
- 测试覆盖率 ≥ 80%
- 支持 3+ 编程语言
- 扫描性能：10,000 文件 < 60s
- 社区 Star > 100
